# PrimeTrade-DataScience-Assignment
Bitcoin Market Sentiment vs Trader Performance Analysis using Python and Google Colab

## Objective
This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and historical trader performance.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib

## Dataset
- Historical Trader Data
- Bitcoin Fear & Greed Index

## Analysis Performed
- Data Cleaning
- Data Merging
- Exploratory Data Analysis (EDA)
- Data Visualization
- Profit & Loss Analysis
- Market Sentiment Analysis

## Key Insights
- Analyzed trading behavior across different market sentiments.
- Compared Buy and Sell trades.
- Identified the most traded cryptocurrencies.
- Evaluated trader profitability under Fear and Greed conditions.
